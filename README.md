# Echoes of The Void
<font-size="1">Public repo of for Echoes of The Void.</font-size>

Echoes of The Void is a 1-4 player fast-paced, 2D top-down "survivor-like" roguelite, where physics based space flight is blended with modern, chaotic item stacking and scaling difficulty.

Blast asteroids, collect items, and run from an ever-increasing horde.
